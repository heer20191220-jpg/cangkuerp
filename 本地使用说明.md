# 仓库 ERP 本地使用说明

## 启动

双击：

`启动联网ERP.bat`

启动后在浏览器打开：

`http://localhost:3000`

同一公司内网的其他电脑可以访问：

`http://服务器IP:3000`

服务器 IP 可以在启动这台电脑上运行 `ipconfig` 查看，一般是 `192.168.x.x` 或 `10.x.x.x`。

## 默认账号

管理员：

- 账号：`admin`
- 密码：`admin123`

录入员：

- 账号：`luru`
- 密码：`123456`

正式给公司使用前，请先修改默认密码：

```powershell
node set-password.js admin 新管理员密码
node set-password.js luru 新录入员密码
```

密码至少 8 位。修改后，旧登录状态会自动失效。

## 当前功能

- 登录
- 新增商品
- 编辑商品
- 删除商品
- 搜索商品
- 导出 CSV
- 自动生成英文品名
- 自动生成英文材质
- 自动生成英文用途
- 自动计算体积
- 自动计算计费重
- 根据中文品名匹配海关编码
- 操作日志记录

## 数据保存位置

当前数据保存在：

`data/db.json`

已自动备份过一次到：

`data/db.backup-before-cleanup.json`

日常备份可以双击：

`backup-data.bat`

备份文件会保存到：

`backups/`

## 注意

当前版本适合作为公司内网试运行版，建议先给少量员工使用，确认字段、流程和导出格式都符合实际业务。

如果后续多人长期同时使用，建议升级到正式数据库，避免 JSON 文件在高并发下成为风险点。
